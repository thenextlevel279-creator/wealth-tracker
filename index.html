Notion import guide

To use your exported CSV files inside Notion:

  1. Download the CSV file for the section you want (Budget, Debts, Income, etc.)
  2. Open Notion and create a new page or open an existing one
  3. Type /table and select Table — Full page or Table — Inline
  4. Click the ... menu on the table → select Merge with CSV
  5. Upload your downloaded CSV file — Notion will auto-populate columns
  6. Adjust column types (Number, Text, Select) as needed in Notion
